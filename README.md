# 🧠 CCTV 이상행동 감지 챗봇 시스템

> 실시간으로 CCTV 이상행동을 감지하고, PostgreSQL DB와 연동하여  
> 이상현상 발생 시 챗봇이 자동으로 대응 메시지를 안내하는 시스템입니다.  
> (Flask + Socket.IO + OpenAI GPT-4o 기반)

---

## 🚀 주요 기능

- 🔍 **실시간 이상현상 감지**
  - PostgreSQL `incidents` 테이블을 주기적으로 모니터링
  - 새로운 이상현상 발생 시 자동으로 챗봇 메시지 푸시 (Socket.IO 이벤트 발생)

- 💬 **자연어 기반 대화 인터페이스**
  - “최근 이상현상 알려줘” → 최근 감지된 이벤트 DB에서 자동 조회
  - 이상유형별 대응 가이드 제공 (흡연, 폭행, 파손, 화재 등)

- 🧠 **LLM 기반 자동 응답**
  - OpenAI **GPT-4o** 모델을 사용하여 상황에 따른 응답 생성
  - 시스템 지침(`system_role`)과 보조지시(`instruction`)를 기반으로 안전 대응 중심 대화 수행
